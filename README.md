# Months Name Java

A simple Java program that displays the name of a month based on a number input.

## Description

This Java application takes a number between `1` and `12` and prints the corresponding month name (e.g., `1 → January`).  
It’s a beginner-level console program useful for learning simple input, control flow, and arrays or switch statements in Java.

## Features

- Displays month name by number
- Simple and clear Java code
- Works for numbers `1` to `12`
